# Sispak CBR - Diagnosa Kerusakan Laptop

Aplikasi sistem pakar berbasis web untuk mendiagnosa kerusakan laptop menggunakan metode **Case-Based Reasoning (CBR)**.

## Fitur
- Login multi role: Admin, Teknisi, User
- Diagnosa kerusakan laptop
- Perhitungan similarity CBR
- Kelola data gejala
- Kelola data kerusakan
- Kelola basis kasus
- Riwayat diagnosa
- Retain case baru
- Setting threshold CBR
- Dashboard statistik
- Export PDF laporan

## Teknologi
- Laravel 10
- PHP 8.1+
- MySQL
- Bootstrap 5
- Chart.js
- DomPDF
